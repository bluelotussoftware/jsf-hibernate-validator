@NotNull

This validator ensures that the input is not null. This does not check to make sure that it is not an empty string.

Note: to make this example work in JSF, you need to set a context parameter in the web.xml configuration to tell JSF to interpret empty Strings as null. 

         <context-param>
              <param-name>javax.faces.INTERPRET_EMPTY_STRING_SUBMITTED_VALUES_AS_NULL</param-name>
              <param-value>true</param-value>
         </context-param>
