Note to make this work in JSF, you need to set a parameter in the configuration to tell JSF to interpret empty Strings as null. 

     <context-param>
          <param-name>javax.faces.INTERPRET_EMPTY_STRING_SUBMITTED_VALUES_AS_NULL</param-name>
          <param-value>true</param-value>
     </context-param>
